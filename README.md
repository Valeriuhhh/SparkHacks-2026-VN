# SparkHacks-2026-VN
Visual Novel called "The Machine" using Renpy Sparkhacks Track 2

## INSTRUCTIONS (for Mac Users)

to run the game on the local computer, you must:
- Download the github repo on your local computer
- double click on the "Ren'Py" application to open the engine
- under projects, select the option "The Machine" 
- click the text "Launch Project" to run the game

## CODE STRUCTURE AND FUNCTIONALITY

- the root directory contains the renpy files for the engine to run, along with the folder titled "The Machine" where all of our main assets for the game are stored
- within the "The Machine" file, it contains the files for the art, backgrounds, UI/UX Design, custom fonts, and the script.rpy file for our main code
